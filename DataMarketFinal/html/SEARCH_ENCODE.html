Description

guess_encoding of data

Usage

SEARCH_ENCODE(file_path)

Arguments

file_path

path of data(file)

Details

Use readr package of guess_encoding. If no probability of encoding more than 20 Otherwise,return most probability of encoding

Value

guess of encoding result

Author(s)

Shunmao

Examples

data(test_data1)
tmp=tempfile(fileext = '.csv')
writeLines(text=test_data1,con=tmp)
SEARCH_ENCODE(tmp)
unlink(tmp)

## The function is currently defined as
function (file_path) 
{
    a = tryCatch(readr::guess_encoding(file_path), error = function(e) return(matrix(NA, 
        1, 1)))
    if (is.na(a[1, 1])) {
        return("unknown")
    }
    else {
        return(a[1, 1] %>% as.character())
    }
  }
