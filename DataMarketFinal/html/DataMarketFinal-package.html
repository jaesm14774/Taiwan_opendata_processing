DataMarketFinal-package {DataMarketFinal}R Documentation

Clean and make data in one format, and also make a tag for this data

Description

Government of data is not unified,so this package want to make data be in one format. Package contains detecting worse & html file function,automatic guessing encoding of file is what,cleaning meaningless row & column, and after doing other processing, return final result.Result maybe clean data or message of can not be dealt.
